# jToast
A modern & easy-going jQuery Plugin to create Toasts.<br><br>

## Demo
[Click](https://l2g.cc/projects/jToast/)
<br>

## Customization options
* Text
* Duration
* Background
* Color
* Border radius
<br>

## Usage & default values
```javascript
showToast(text, {
  duration: 3000,
  background: '#232323',
  color: '#fff',
  borderRadius: '0px'
});
```
<br>

## An example
```javascript
showToast('Thanks for signing up!', {
  duration: 5000,
  background: '#20b2aa',
  borderRadius: '15px'
});
